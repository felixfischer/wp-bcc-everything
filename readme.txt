Automatically set Admin Bcc on every outgoing email. Useful for verifying that emails get sent correctly.

== Description ==

This plugin automatically sets Admin Bcc on every outgoing email. This uses the main blog email address as defined in
Settings -> General. There is nothing to configure in this plugin. Just activate and forget.

== Installation ==

1. Upload `wp-bcc-everything.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
